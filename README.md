# 개인용 포트폴리오 웹사이트

## 기술 스택
- React
- TypeScript
- Tailwind CSS
- Framer Motion

## 프로젝트 개요
개인용 포트폴리오 웹사이트를 제작했습니다.

## 프로젝트 구조
- `src/components`: 재사용 가능한 컴포넌트
- `src/sections`: 섹션별 컴포넌트
- `src/assets`: 이미지 및 기타 에셋
- `src/styles`: 스타일 관련 파일
- `src/types`: 타입 정의
- `src/utils`: 유틸리티 함수

## 실행 방법
```bash
npm install
npm run dev
```

## 배포
```bash
npm run build
npm run deploy
```
